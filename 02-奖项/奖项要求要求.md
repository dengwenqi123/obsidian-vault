
## 一、评选目的及原则

1. **目的**：为搭建小米手机部流程和信息化建设的积极氛围，并表彰为此做出突出贡献的团队。为规范小米手机部流程信息化奖申报、评选、表彰和管理工作，特制定本方案
    
2. **原则**：优中选优，宁缺毋滥、识别真正有价值的优秀项目。奖项数量根据实际提报项目及信息的质量酌情调整
    

## 二、评选周期及范围

1. **评选周期：**年度奖项，即年/次
    
2. **评选范围**：本评选面向集团内围绕手机部流程和信息化建设的相关团队及个人，以项目为单位申报，包含参与手机部业务流程信息化建设的团队或个人
    

## 三、奖项设置及评选方式

1. **奖项定义**
    
    1. **流程类：**
        
        1. 旨在手机部流程管理组立项的流程建设项目
            
        2. 参与评选时应完成项目90%及以上的工作进度，或者完成90%及以上的项目目标
            
    2. 信息化类：
        
        1. 旨在手机部信息化管理组立项通过的信息化建设项目；
            
        2. 参与评选时完成结项与复盘（跨年重大项目必须在申报年度结项），是衡量项目实际产出和价值的主要依据
            
2. **奖项设置**
    
    |   |   |   |
    |---|---|---|
    |**奖项**|**数量**|**激励方式**|
    |**流程类项目奖**|2个|50,000元/个|
    |**信息化类项目奖**|2个|50,000元/个|
    
3. **申报规模**
    
      **千人以上一级部门：**流程/信息化类通过初选**上限为两个**，如果被提名项目不达要求初选不通过，名额可空缺。
    
      **千人以下一级部门：**流程/信息化类通过初选**上限为一个**，如果被提名项目不达要求初选不通过，名额可空缺。

## 五、评选流程

暂时无法在飞书文档外展示此内容

1. **申报****：**由流程管理部根据年度报奖计划发送申报通知，同时各部门启动遴选优秀项目。
    
2. **补录**：补录并非第二次申报机会，是为在4月份申报时尚未立项的项目增开的机会。评委会评估补录原因，对于合理的项目才可释放参评资格。
    
3. **提名****/收集材料**：由各部门报奖接口人按照报奖模板要求，在截止日期内收集本部门的报奖，并上报流程部相应接口人。答辩会上的宣讲材料需要和提交初选的内容一致。
    
4. **初筛：**
    
      奖项主办方组织相关方根据提报的项目成果以及价值，是否重复申报、是否当期成果、是否有重大负面事件等，结合评选标准进行初选打分评选出入围答辩名单。
    
5. **答辩：**
    
    1. 对候选人要求：抽签产生答辩顺序，答辩人应清晰表达申报项目对成本的贡献及核心价值和关键亮点。
        
    2. 对评委要求：评委由手机部流程及信息化管理组成员担任；每次答辩评审会必须有不少于7名（含7名）评委参加评审，不能按照会邀参加答辩评审会的，需要提前24小时请假；评委不参与本部门申报的项目及个人奖项打分，评委不能帮答辩人回答问题。
        
    3. 时间要求：答辩时间10分钟/人(项目派1名代表)，评委提问5分钟。
        
6. **结果及公示****：**评委进行打分，去掉一个最低和一个最高分后，算平均分按照流程类和信息化类分别进行排序评优，形成评选结果；结果向全员公示1周，如无异议，报批手机部流程与信息化领导小组主任审批最终获奖结果。
    
7. **表彰：**表彰活动由手机部平台管理组织统筹实施。
